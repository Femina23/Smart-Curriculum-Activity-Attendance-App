# 📚 Smart Curriculum Activity & Attendance App

A responsive web application designed to simplify curriculum activity management and attendance tracking for students and faculty. The system provides an easy-to-use interface for managing academic activities, viewing attendance records, and generating reports.

---

## 🚀 Features

- 👨‍🎓 Student Login
- 👨‍🏫 Faculty Login
- 📅 Attendance Management
- 📖 Curriculum Activity Tracking
- 📊 Report Generation
- 📱 Responsive User Interface
- 🔐 Simple Authentication System

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
